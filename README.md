# Personal Portfolio Website

A complete, responsive portfolio website built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` - page structure and content sections
- `styles.css` - design system, layout, and responsive styles
- `script.js` - menu toggle, reveal animations, footer year, and demo contact form behavior

## Sections Included

- Hero
- About
- Skills
- Projects
- Experience Timeline
- Contact Form

## Quick Start

1. Open `index.html` in your browser.
2. Edit your name, bio, project details, and experience in `index.html`.
3. Adjust colors and typography from CSS variables in `styles.css`.

## Personalization Checklist

- Replace `Your Name` in headings, footer, and title.
- Update project cards with your real work and links.
- Add your social links (GitHub, LinkedIn, X) in header or footer.
- Connect the contact form to a real backend service (e.g. Formspree, Netlify Forms, or custom API).

## Deploy

You can deploy this site on:

- GitHub Pages
- Netlify
- Vercel

No build step is required.
